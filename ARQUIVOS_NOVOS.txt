FUNCIONALIDADES IMPLEMENTADAS:

ÁREA DO MORADOR:
✅ Login do Morador (CPF + Senha)
✅ Edição de Cadastro (email, telefone, celular, CPF)
✅ Gerenciamento de Veículos (adicionar, editar, excluir)
✅ Visualização de Protocolos (pendentes e entregues)
✅ Dados do Hidrômetro (número, lacre, última leitura)
✅ Histórico de Leituras (data, leitura anterior, atual, consumo, valor)
✅ Notificações (visualizar, marcar como lida, baixar anexos)

ÁREA ADMINISTRATIVA:
✅ Cadastro de Notificações (data/hora, assunto, resumo, anexo)
✅ Upload de Anexos (PDF, JPG, PNG até 10MB)
✅ Edição e Exclusão de Notificações
✅ Relatórios Detalhados (visualizações e downloads por morador)
✅ Número Sequencial Automático
✅ Acompanhamento de Interações dos Moradores

INSTRUÇÕES DE INSTALAÇÃO:
1. Extraia o arquivo ZIP no servidor
2. Execute o script database_notificacoes.sql no banco de dados
3. Crie o diretório: mkdir -p uploads/notificacoes && chmod 755 uploads/notificacoes
4. Copie todos os arquivos para o diretório raiz do sistema
5. Acesse login_morador.html para testar área do morador
6. Acesse notificacoes.html para testar área administrativa
