INSTRUÇÕES DE INSTALAÇÃO - MÓDULO CHECKLIST VEICULAR
Sistema de Controle de Acesso - Serra da Liberdade
================================================================================

PASSO 1: EXECUTAR O SCRIPT SQL
-------------------------------
1. Acesse o phpMyAdmin ou MySQL via terminal
2. Selecione o banco de dados: inlaud99_erpserra
3. Execute o arquivo: database_checklist.sql
4. Verifique se as 5 tabelas foram criadas:
   - checklist_veicular
   - checklist_itens
   - checklist_alertas_config
   - checklist_alertas_gerados
   - checklist_km_acumulado

PASSO 2: VERIFICAR ARQUIVOS
----------------------------
Certifique-se de que os seguintes arquivos estão no servidor:

APIs (Backend):
- api_checklist.php
- api_checklist_itens.php
- api_checklist_alertas.php

Páginas (Frontend):
- checklist_veicular.html
- checklist_novo.html
- checklist_preencher.html
- checklist_fechar.html
- checklist_visualizar.html
- checklist_alertas.html

SQL:
- database_checklist.sql

PASSO 3: VERIFICAR INTEGRAÇÃO
------------------------------
1. Acesse: manutencao.html
2. Verifique se o card "Checklist Veicular" aparece
3. Verifique se o link no submenu está funcionando

PASSO 4: TESTAR O SISTEMA
--------------------------
1. Acesse: checklist_veicular.html
2. Clique em "Novo Checklist"
3. Verifique se os veículos aparecem (do módulo de abastecimento)
4. Verifique se os operadores aparecem (do cadastro de usuários)
5. Crie um checklist de teste
6. Preencha os itens
7. Feche o checklist
8. Visualize o checklist criado

PASSO 5: CONFIGURAR ALERTAS (OPCIONAL)
---------------------------------------
1. Acesse: checklist_alertas.html
2. Ajuste os KM limite conforme necessário
3. Ative/desative alertas conforme preferência

OBSERVAÇÕES IMPORTANTES
-----------------------
- O sistema usa as tabelas existentes:
  * abastecimento_veiculos (para veículos)
  * usuarios (para operadores)
  
- Certifique-se de que há:
  * Pelo menos 1 veículo cadastrado em Abastecimento
  * Pelo menos 1 usuário ativo no sistema

- O sistema já vem com 10 alertas pré-configurados

VERIFICAÇÃO DE SUCESSO
----------------------
✓ Tabelas criadas no banco de dados
✓ Arquivos copiados para o servidor
✓ Card aparece na página de Manutenção
✓ Consegue criar um checklist
✓ Consegue preencher itens
✓ Consegue fechar checklist
✓ Consegue visualizar e imprimir
✓ Alertas são gerados automaticamente

SUPORTE
-------
Em caso de problemas:
1. Verifique os logs de erro do PHP
2. Verifique o console do navegador (F12)
3. Verifique se o banco de dados está acessível
4. Verifique as permissões dos arquivos
